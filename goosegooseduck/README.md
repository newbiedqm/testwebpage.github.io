# 鹅腿阿姨模拟器

副标题：《今天卖什么腿》

一个像素风网页端模拟经营小游戏。玩家通过一张张事件卡经营校门口夜宵摊，在现金、信任、合规、流量、体力之间做取舍，并根据隐藏叙事变量进入不同结局。

## 运行

直接打开 `index.html` 即可游玩。也可以用本地静态服务：

```bash
python3 -m http.server 8765
```

然后访问 `http://localhost:8765/`。

## 操作

- 点击选项推进剧情。
- 键盘 `1` 到 `4` 可快速选择。
- 显示模式可在 `自动 / 手机 / 电脑` 之间切换。
- 进度会保存在浏览器 `localStorage` 中。
